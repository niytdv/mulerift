# Assets Required for FluidGlass Component

## 3D Models (GLB format)
Place these files in `public/assets/3d/`:
- `lens.glb` - Cylindrical lens shape
- `cube.glb` - Cube shape
- `bar.glb` - Bar/rectangular shape

You can create these using Blender or download from:
- https://sketchfab.com/
- https://poly.pizza/
- https://market.pmnd.rs/

## Demo Images (WebP format)
Place these files in `public/assets/demo/`:
- `cs1.webp`
- `cs2.webp`
- `cs3.webp`

These are used for the scrolling image gallery in the FluidGlass component.

## Temporary Solution
The FluidGlass component is currently disabled until you add these assets.
The home page will show without errors but won't display the 3D glass effect.
